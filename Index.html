The Caligo Project

Today, most businesses are moving thier operations to the cloud because of efficiency and cost concerns. Cloud computing represents a brave new world where near instantaneous data analysis coupled with an almost unlimited storage capacity will allow developers to build resilient platforms that translate into increased productivity and profit for their business.

Hybrid Cloud Architecture

The Caligo Project is a suite of tools designed to seemlessly facilitate cooperation between cloud services and your business network. The hybrid concept stratifies the cloud system into two layers: Cloud servers and office servers. Each layer is responsible for a distinct operations yet designed to work together for the greatest impact. Below, is a list of sub-projects that make up the Caligo project. You can browse each project or begin the quick-start tutorial for a comprehensive view of the system and how to use it.
